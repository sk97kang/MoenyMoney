* 공통기능 *

- 프로그램 아이콘 표시
- 실시간 시계 기능
- 알림 ON-OFF 기능 (채팅 구현 할 때 기능 보여줌)
- MenuBar 마우스 Enter-Exit에 따른 상태 변경
- MenuBar ToolTip 적용
- MenuBar 현재 Menu 표시 기능
- 미로그인시 MenuBar 사용 불가능

* Home Page 기능 *

- 로그인 기능
 => 필터(ID/PW 불일치, ID/PW 미입력)
 => 대소문자 구분
 => 한글 ID 지원
- 회원가입 기능
 => 필터(ID 중복, ID/PW 미입력)
 => 대소문자 구분
 => 한글 ID 지원

- 계좌정보 기능
 => 총 잔액 표시
 => 계좌개수에 따른 은행, 계좌번호, 계좌잔액 표시
- 로그아웃 기능 (마지막에 실현)

* Input Page 기능 *

- Input 기능
 => 날짜 힌트 기능
 => 필터 기능(Date/Account Format 불일치)
 => 사용자에 따른 계좌 표시
 => 입금/출금 토글 기능
 => 입금/출금에 따른 분류 표시
 => 금액은 숫자만 가능

* Record Page 기능 *

- 사용자에 따른 전체 데이터 출력
- 1주일, 1개월, 6개월 기간별 데이터 출력
- 상세조회 기능
 => 입금/출금 각각의 분류에 따른 데이터 출력
- 입금/출금 각각의 통계 기능

* Account Page 기능 *

- 계좌 추가 기능
 => 계좌번호 추가 힌트 기능
 => 새 계좌 추가 하면 Home Page 에 바로 업데이트 (은행이름, 계좌번호 표시)
- 계좌 삭제 기능
- 계좌 추가 시 필터(계좌번호 미입력, 계좌번호 개수 초과, 계좌번호 중복)

* Chat Page 기능 *

- 전체 사용자 간 채팅 지원
- 자신의 메시지는 오른쪽 정렬, 연하게 표시
- 타인의 메시지는 왼쪽 정렬, 진하게 표시
- 채팅을 하고있다가 다른 페이지 이동시 알림 기능
- 알림 메시지에서 채팅방으로 바로 이동 버튼 지원
- 알림 ON-OFF 지원
- 명령어 지원
 => @계산기 : 계산기 실행
 => @현재잔액 : 현재 잔액 표시
